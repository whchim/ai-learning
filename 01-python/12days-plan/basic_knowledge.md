# Python 基础入门

## 一、数据存储与运算

### 1.字面量

- 字面量：指程序中，直接书写的固定值（数据），就称位字面量。

  ​																	**字面量类型及书写格式**

  | 类型            | 说明                            | 示例                                      |
  | --------------- | ------------------------------- | ----------------------------------------- |
  | 数字类型        | 整数 (int)                      | `10`, `18`, `-5`, `0`                     |
  |                 | 浮点数/小数 (float)             | `8.5`, `3.14`, `1.0`, `-3.5`              |
  | 布尔 (bool)     | 表达现实生活中的逻辑，真或假    | `True`, `False`                           |
  | 字符串 (str)    | 描述文本的一种数据类型          | `"人生苦短，我用Python"`                  |
  | 空值 (NoneType) | 表示空或无值，仅包含一个值 None | `None`                                    |
  | 数据容器        | 存储多项数据的容器              | 列表 / 元组 / 集合 / 字典（后面详细讲解） |

  说明：布尔类型（`bool`）本质就是数字类型。在涉及数学运算时，Python 会自动将Ture转为1，False转为0。

### 2.变量

- 变量：程序中用来存储单个数据的容器，通常会把经常发生变化的数据存储在变量中。

- 定义格式：变量名 = 变量的值    例如：num = 1314.1

  说明：Python是动态类型语言，在程序运行时才进行类型检查，变量的类型可以在程序运行过程中改变（一个变量可以接收不同类型的值）。


### 3.标识符

- 标识符 是程序员在代码中为变量、函数、类等元素所起的名字

- 命名规则（规定）：

  1.只能包含字母（a-z，A-Z）、数字（0-9）、下划线（_）

  2.不能以数字开头

  3.不能使用关键字：True、False、None、and、or、if、else、elif、for、while等

  4.严格区分大小写，比如：age，Age，AGE是三个变量

  ```
  正确:
  name Name -name __name na_me name_ true name6 naME
  错误：
  na$me and or False 6name
  ```

  命名规范（变量）：

  1.见名知意

  2.多个部分使用下划线连接

  3.英文字母全小写

  ```
  正确：
  name age color update_time my_name user_first_name (蛇形命名法)
  错误：
  a a1 a2 updatetime myname
  ```

​     PEP8：PEP是Python社区的核心技术文档和标准化机制，而PEP8是Python的代码风格指南（<https://peps.python.org/pep-0008>）

### 4.数据类型

- Python中常见的基础数据类型：

  | 标准名称 |  描述  | 说明                                         |
  | :------: | :----: | -------------------------------------------- |
  |   int    |  整数  | 数字类型，存放整数，如：10、-5               |
  |  float   | 浮点数 | 数字类型，存放小数，如：8.5、3.14、1.0、-3.5 |
  |   str    | 字符串 | 用引号引起来的都是字符串，如："Python"       |
  |   bool   |  布尔  | 布尔类型，描述真和假，如：True、False        |
  | NoneType |  空值  | 表示空或无值，仅包含一个值 None              |

  1.通过type()语句来得到数据的类型，具体语法为：**type(要查看类型的数据)**
  
  ```
  print(type("Hello")) # 类型：str
  print(type(10)) # 类型：int
  print(type(3.14)) # 类型：float
  print(type(True)) # 类型：bool
  print(type(None)) # 类型：NoneType
  ```
  
  2.通过isinstance()检查数据是否属于指定的类型，返回的是一个bool值，即如果是：True，否则：False，具体语法为：isinstance（数据、类型）
  
  ```
  num = 5.0
  print(num)
  print(isinstance(num,int)) # False
  ```
  
  说明：变量本身是没有类型的，tpye（变量）输出的类型是变量中存储的数据的类型。

### 5.字符串定义

- 字符串的三种定义方式：

  ```python
  # 双引号定义
  s1 = "Hello"
  # 单引号定义
  s2 = 'Python'
  # 三引号定义（多行字符串）
  s3 = """
  尊敬的客户：
      感谢您选择我们公司的产品。
      我们会为您竭诚的服务。
      祝好 ~
  """
  ```

  使用原则：单引号 与 双引号是等效的，项目中保持一种写法即可

- 常见的转义字符

  | 转义字符 | 名称   | 作用                        |
  | -------- | ------ | --------------------------- |
  | `\'`     | 单引号 | 表示字面量的单引号 `'`      |
  | `\"`     | 双引号 | 表示字面量的双引号 `"`      |
  | `\n`     | 换行符 | 开始新的一行（换行）        |
  | `\t`     | 制表符 | 增加缩进，相当于一个 Tab 键 |

字符：是文本世界的基本单位，一个字母、一个数字、一个标点符号、一个汉字等都是一个字符

### 6.字符串拼接

- 很多时候，我们需要将多个字符串拼接起来，可以直接使用（+）进行拼接。【易出错，不推荐在复杂场景使用】

  ```python
  slogan = "姓名" "张三”
  print(slogan)
  
  slogan = "姓名" + "张三"
  print(slogan)
  
  s1 = "人生苦短"
  s2 = "我用Python"
  
  print("里昂·斯科特·肯尼迪:" + " , " + s2)
  ```

  注意：+ 号可以用来拼接两个字符串，但是无法将非字符串与字符串进行拼接（非字符串类型需要转换为字符串类型）。

### 7.字符串格式化

- 通过 `%占位符` 的形式完成字符串和变量的快速拼接。（其中 `%` 表示我要占位，`s` 表示将变量转为字符串放入占位的位置）。【传统写法，兼容性好但不够灵活】

  ```python
  s1 = "东哥"
  
  print("大家好，我是%d，欢迎大家进入Python课程" % s1)
  
  s1 = "人生苦短"
  s2 = "我用Python"
  
  print("里昂·斯科特·肯尼迪: %s , %s" % (s1,s2))
  ```

  注意：前面有多少个占位符（%s），后面就需要有多少个变量(或数据)，前后数量需要一致。

- 也可以通过 **f"内容{变量/表达式}"** 的形式来完成快速格式化。【最现代、简洁、可读性强，Python 3.6+ 推荐使用】

  ```python
  name = "东哥"
  print(f"大家好，我是{name}，欢迎大家进入Python课程的学习")
  
  s1 = "人生苦短"
  s2 = "我用Python"
  
  print(f"里昂·斯科特·肯尼迪:{s1},{s2}")
  ```
