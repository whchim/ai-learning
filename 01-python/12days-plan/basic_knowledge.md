# Python 基础入门

## 一、数据存储与运算

### 1.字面量

- 字面量：指程序中，直接书写的固定值（数据），就称位字面量。

  ​																	**字面量类型及书写格式**

  | 类型            | 说明                            | 示例                                      |
  | --------------- | ------------------------------- | ----------------------------------------- |
  | 数字类型        | 整数 (int)                      | `10`, `18`, `-5`, `0`                     |
  |                 | 浮点数/小数 (float)             | `8.5`, `3.14`, `1.0`, `-3.5`              |
  | 布尔 (bool)     | 表达现实生活中的逻辑，真或假    | `True`, `False`                           |
  | 字符串 (str)    | 描述文本的一种数据类型          | `"人生苦短，我用Python"`                  |
  | 空值 (NoneType) | 表示空或无值，仅包含一个值 None | `None`                                    |
  | 数据容器        | 存储多项数据的容器              | 列表 / 元组 / 集合 / 字典（后面详细讲解） |

  说明：布尔类型（`bool`）本质就是数字类型。在涉及数学运算时，Python 会自动将Ture转为1，False转为0。

### 2.变量

- 变量：程序中用来存储单个数据的容器，通常会把经常发生变化的数据存储在变量中。

- 定义格式：变量名 = 变量的值    例如：num = 1314.1

  说明：Python是动态类型语言，在程序运行时才进行类型检查，变量的类型可以在程序运行过程中改变（一个变量可以接收不同类型的值）。


### 3.标识符

- 标识符 是程序员在代码中为变量、函数、类等元素所起的名字

- 命名规则（规定）：

  1.只能包含字母（a-z，A-Z）、数字（0-9）、下划线（_）

  2.不能以数字开头

  3.不能使用关键字：True、False、None、and、or、if、else、elif、for、while等

  4.严格区分大小写，比如：age，Age，AGE是三个变量

  ```
  正确:
  name Name -name __name na_me name_ true name6 naME
  错误：
  na$me and or False 6name
  ```

  命名规范（变量）：

  1.见名知意

  2.多个部分使用下划线连接

  3.英文字母全小写

  ```
  正确：
  name age color update_time my_name user_first_name (蛇形命名法)
  错误：
  a a1 a2 updatetime myname
  ```

​     PEP8：PEP是Python社区的核心技术文档和标准化机制，而PEP8是Python的代码风格指南（<https://peps.python.org/pep-0008>）

### 4.数据类型

- Python中常见的基础数据类型：

  | 标准名称 |  描述  | 说明                                         |
  | :------: | :----: | -------------------------------------------- |
  |   int    |  整数  | 数字类型，存放整数，如：10、-5               |
  |  float   | 浮点数 | 数字类型，存放小数，如：8.5、3.14、1.0、-3.5 |
  |   str    | 字符串 | 用引号引起来的都是字符串，如："Python"       |
  |   bool   |  布尔  | 布尔类型，描述真和假，如：True、False        |
  | NoneType |  空值  | 表示空或无值，仅包含一个值 None              |

  1.通过type()语句来得到数据的类型，具体语法为：type(要查看类型的数据)
